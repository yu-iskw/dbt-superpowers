# Read Metrics

Skill for discovering and inspecting metrics in the Lightdash semantic layer.

## Purpose

Enables identification and inspection of existing metrics for modeling and analysis purposes.

## Tools

Wraps the following MCP tools from the `lightdash` server:

- `ldt__list_metrics`

## Behavior

1. **Discovery**: Use `list_metrics` to find available metrics for a project.

## Rules

- Use `read-metrics` to understand current aggregation logic before proposing changes.
